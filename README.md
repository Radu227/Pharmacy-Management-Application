\# Pharmacy Management Application



O aplicație desktop dezvoltată în \*\*C++\*\* pentru gestionarea eficientă a stocurilor de medicamente dintr-o farmacie.



\## Funcționalități Principale

\* \*\*Gestiune Stoc:\*\* Operații CRUD complete (Adăugare, Ștergere, Modificare, Afișare medicamente).

\* \*\*Filtrare \& Sortare:\*\* Căutare rapidă după denumire, substanță activă sau preț.

\* \*\*Undo/Redo:\*\* Funcționalitate de anulare sau refacere a ultimelor acțiuni efectuate.

\* \*\*Arhitectură OOP:\*\* Implementare bazată pe principii solide de încapsulare și separare a responsabilităților.



\## Detalii Tehnice

\* \*\*Limbaj:\*\* C++

\* \*\*Interfață:\*\* Grafică (GUI) / Consola (în funcție de versiune)

\* \*\*Persistență:\*\* Datele sunt salvate și încărcate din fișiere externe.
